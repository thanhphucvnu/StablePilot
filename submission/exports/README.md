# StablePilot — logo exports (submission)

| File | Suggested use |
|------|----------------|
| `stablepilot-logo.svg` | Vector master (any size, web, print). |
| `stablepilot-logo-192.png` | Favicon / small avatar (~192×192). |
| `stablepilot-logo-512.png` | **DoraHacks / social thumbnail** (~512×512). |
| `stablepilot-logo-1024.png` | High-res icon, slides, or crop for banners (~1024×1024). |

**Regenerate** from repo root:

```bash
cd galactica-agent && node scripts/export-logo.mjs
```

Source artwork: `galactica-agent/public/logo.svg`.
